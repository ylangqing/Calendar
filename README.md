# ics
Calendar

## 使用方法
以下提供两种导入该日历的方法。

### 1. 直接导入日历
直接下载`.ics`文件，然后导入到日历APP中。

### 2. 使用订阅URL
复制以下URL，然后添加到相应的日历APP中。
CQF英文课程日历：
```
https://raw.githubusercontent.com/ylangqing/Calendar/ics/CQFCalendar.ics
```
CQF中文课程日历：
```
https://raw.githubusercontent.com/ylangqing/ics/calendar/CQFCalendar_Chinese.ics
```
